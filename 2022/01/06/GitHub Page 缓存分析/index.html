GitHub Page 缓存分析

文章通不过去之后，页面刷新不过来。

1、如果 Active 是正常的，就是没有刷新过来，通过清除浏览器的缓存了来解决。

2、还要注意有时候是编译错误，错误的话 GitHub 会给你的邮件发信息，目前我没有复现过。
